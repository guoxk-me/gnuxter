## What is included

This starter now ships with three content collections:

- `docs` for guides and reference material
- `blog` for posts and design notes
- `changelog` for release history

English content lives on the default route space, so this file resolves under `/docs/getting-started`.

## What comes next

The next logical layer is to add:

1. `@nuxtjs/i18n` for true locale routing
2. `@nuxtjs/seo` for sitemap, robots, and metadata
3. richer page chrome like sidebars, table of contents, and search

> The goal is not a tiny starter anymore. It is a content-first base that can intentionally grow heavy.
